# SeaSharpTextGame
Untitled Text Game

SeaSharpTextGame is a 1v1 table top text based battle simulator 
Take turns attacking the enemy and see if you can win.

More coming soon.


# Setup 

### Windows 
Required .Net Core 3.1 
A Text Game

## Linux 
Tested on Debian 12

```bash
wget https://dot.net/v1/dotnet-install.sh -O dotnet-install.sh
```
